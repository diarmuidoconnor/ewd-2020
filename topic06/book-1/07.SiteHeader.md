## SiteHeader.

We need a more elegant site-wide header for the app. Crete the folder `components/siteHeader` and add these files:

+ siteHeader.css:
~~~
.fixed-top {
    padding-top: 0px;
    padding-bottom: 0px;
}
~~~

+ index.js:
~~~
import React from "react";
import { Link } from "react-router-dom";
